# Sheabok

**Bakuchiol + Shea Butter Anti-Aging Cream**

A luxury marketing website for Sheabok — a premium anti-aging cream powered by bakuchiol and shea butter.

## Live Site

[https://freddyfarbod.github.io/Sheabok-dev-public/](https://freddyfarbod.github.io/Sheabok-dev-public/)

## Overview

Single-file HTML marketing site with a white and gold luxury aesthetic. No build tools, no frameworks — just clean HTML, CSS, and vanilla JavaScript.

### Features

- **Responsive design** — fluid typography and mobile-first layout
- **Interactive elements** — mouse-tracking spotlight, parallax product card, scroll animations, FAQ accordion, quantity selector
- **Accessibility** — semantic HTML, ARIA labels, keyboard navigation, reduced motion support
- **Sections** — Hero, Benefits, Ingredients, How to Use, Reviews, FAQ, Buy

### Design

- Soft ivory/off-white backgrounds with elegant gold (#C6A75E) accents
- Cormorant Garamond (display) + Outfit (body) typography
- Subtle shimmer animations on gold buttons
- Thin gold gradient dividers between sections
- Soft shadows for depth and premium feel

## Setup

No installation needed. Open `index.html` in a browser or deploy via GitHub Pages.
